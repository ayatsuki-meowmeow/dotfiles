# Global CLAUDE.md

このファイルは全プロジェクト横断で読み込まれるユーザーレベル設定です。
dotfilesリポジトリで管理しており、`~/.claude/CLAUDE.md` にsymlinkされています。

## 開発スタイル / 好み

- 会話は全て日本語にしてください
- モデルの使い分け
  - **Opus**: 思考、設計、プラン作成、複雑な判断を伴うタスク
  - **Sonnet**: 実装タスク、定型的な作業パターン、決まったコード生成
- 1タスクで変更する diff は 200 行以下を目安にしてください
  - カウント対象: 手書きの実装コード（追加+変更行の合計）
  - 除外: テスト、自動生成ファイル、lock ファイル、マイグレーション、純粋なリネーム/一括置換
  - 超過が見込まれる場合は、実装着手前にタスク粒度の再設計について相談してください

## やってほしくないこと

- 下記ファイルへのアクセス、および内容を閲覧する行為（log 出力・ファイル読み込み・コマンド経由など、手段を問わず）は禁止です
  - `.env`
  - `.env.*` （`.env.local`, `.env.production` など全て対象）
  - `.envrc`
- 依頼されたタスクの範囲を超えた作業を勝手に実行することは禁止です
  - 例: コードの最適化、テストの追加・修正、リファクタリング、コメントの追加、未使用コードの削除、依存パッケージの更新
  - これらは私が明示的に依頼した場合のみ実行してください。「ついで」に行うのは禁止です
  - 判断に迷う場合は実行前に確認してください
